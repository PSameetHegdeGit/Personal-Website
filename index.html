Projects

News Aggregator

This news aggregator uses a external api NewsAPI to aggregate news sources by keyword. I developed a frontend in Angular and a backend in Flask. Both are deployed on Heroku. Rather than communicating via HTTP between the frontend and api directly, I chose a Flask backend in order to perform some analytics on the data and communicate with a MongoDB database (I use to store some user information). Additionally, although NewsAPI sends data in a JSON format anyway, I wanted to reformat the data so that it would be easier to render this data into html.

Angular | Flask | MongoDB | NewsAPI

Simplex

This project finds a vector that gives the optimal solution to a linear optimization problem. The algorithm follows the traditional simplex algorithm and is programmed in python. The backend runs on a flask server deployed to Heroku. I developed a minimalistic user interface with vanilla Javascript, HTML, and CSS. The frontend is deployed as a static website on AWS.

More information on the simplex algorithm can be found here: https://www.matem.unam.mx/~omar/math340/simplex-intro.html

Javascript | HTML/CSS | Flask

Modeling Hardy Weinberg

This Unity game simulates the frequency of specific genotypes in a population over generations. In this world, we make several assumptions of the environment as well as the population such that the population's genotypic frequencies in some later generation stays consistent. This consistency is the Hardy Weinberg equilibrium. The Unity game is coded with c# and communicates with a Flask backend. This Flask backend takes several parameters from the game and uses these values to determine a random genotypic frequency and the changes in this genotypic frequency over time (these frequencies eventually reach the Hardy Weinberg Equilibrium).

More information on the Hardy Weinberg equilibrium can be found here: https://www.nature.com/scitable/definition/hardy-weinberg-equilibrium-122/

Unity | C# | Flask

CapSheet Automation
